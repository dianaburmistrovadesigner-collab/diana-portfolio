### 01 · DocuChain
**Secure E-Signature + Crypto-Backed Audit Trail**

| | |
|---|---|
| **Тип** | Web app (mobile-responsive) |
| **Бейдж** | 0→1 MVP |

**Контекст:** «Upload → sign → done» simplicity with a verifiable integrity layer anchored to a public blockchain — without forcing users to understand blockchain.

**Мой вклад:**
- Designed the core signing flow: upload, add recipients, review & send, signing view, success state
- Created clear step-based progress indicators to reduce anxiety in a 'legal-ish' flow
- Defined 'trust moments' UX: what users need to understand vs. what stays optional/advanced

**Демонстрирует:** 0→1 product UX for a trust-heavy workflow, progressive disclosure, and keeping a 'crypto security' benefit user-friendly.
