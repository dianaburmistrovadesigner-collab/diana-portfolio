### 01 · DocuChain
**Secure E-Signature + Crypto-Backed Audit Trail**

| | |
|---|---|
| **Тип** | Web app (mobile-responsive) |
| **Бейдж** | 0→1 MVP |

**Контекст:** Простота «загрузил → подписал → готово» с верифицируемым слоем целостности на публичном блокчейне — без необходимости разбираться в крипте.
